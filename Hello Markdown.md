#### 创建更新目录

安装完“Markdown All in One扩展后，你可以通过以下步骤生成 Markdown 目录：

打开你的 Markdown 文件。
将光标放置在你想要插入目录的位置。
按 Ctrl+Shift+P 打开命令面板。
输入或选择 “Create Table of Contents” 命令。
扩展会根据你文档中的标题（如 #, ##, ### 等）自动创建一个目录。
